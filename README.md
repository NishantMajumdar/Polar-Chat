# Polar-Chat
## Features
* Nice animations
* Bad words filter
* Youtube videos embedding
* User typing display
* Custom rooms with code
* Default public room
* Private messages
* Voice calling
* Text commands
* Admins
* Anti spam
* User mention
* Can send images
* Script attack protection
* Username changer
* And much more
